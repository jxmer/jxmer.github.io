python3 学习爬虫遇到的一点编码问题

发表于 2017-07-02 | 分类于 python | 阅读次数:

在windows上默认以gbk编码。保存一个文件到本地，会被改成gbk的编码，即使在输出内容在写入到文件前已经做了utf-8的编码，最后还是会被转成gbk编码。
本例中就是从糗百上抓的文本，其本身是utf-8编码的。读取的时候encode用gbk，输出用gbk。既然最后一定要用gbk，那前面就不可以用utf-8来encode，否则会报错，i.e.，有些字符找不到对应的编码。

北邮人能登陆，但是utorrent不能下载的解决方法

发表于 2018-03-17 | 阅读次数:

这是win10的DNS解析发生了变化，找不到tracker.byr.cn。出现上述问题，在utorrent里面也能够看清楚的，就是在下载任务的tracker找不到。这样问题就好办了，解决问题的方法就是在hosts文件里面添加两行

2001:da8:215:4078:250:56ff:fe97:654d bt.byr.cn
2001:da8:215:4078:250:56ff:fe97:654d tracker.byr.cn
